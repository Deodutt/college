Player Unkown Battle Grounds

PUBG: $29.99

PLAYERUNKNOWN'S BATTLEGROUNDS is a multiplayer online battle royale game developed by PUBG Corporation. This is a survival game that is essentially last man standing wins. There is an competitive side to this game that host tournaments very often. There is a large variety of guns that are scattered around the map and players get to drop out a plane to find these guns to kill other players. The Winner gets a chicken dinner.



PLAYERUNKNOWN'S BATTLEGROUNDS is playable on Xbox One, Android, iOS, and Microsoft Windows.
