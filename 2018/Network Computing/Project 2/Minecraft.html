Minecraft

Minecraft is basically a sandbox game that was developed by Mojang. In this game you can do anything you want. You can build, fight, roleplay, etc. There is no object of this game. There are different servers that you can join to have fun. Every server is a different experience.

Minecraft cost $25 dollars to play.
