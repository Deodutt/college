Notable Projects

Natours Project

The Natours Project is created by Jonas Schmedtmann. Jonas provides video tutorials of this entire project on UDEMY. This project is a great way to learn a ton of new CSS features, build amazing animation effects, and become familiar with advanced techniques to create professional pages.

Trillo Project

The Trillo Project is created by Jonas Schmedtmann. Jonas provides video tutorials of this entire project on UDEMY. The Trillo project teaches users how to make their lives easier while doing CSS. Jonas teaches his students the fundemental concepts and properties of Flexbox using SASS. You will also learn some custom CSS properties, masks, and SVGs.
